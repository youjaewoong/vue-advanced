# chart-lib
```
- vue create chart-lib
- npm install chart.js@2

1. 차트 라이브러리 NPM으로 설치
2. 설치된 라이브러리를 import로 App.vue에서 로딩
3. mounted() 라이프 사이클 훅에서 차트를 그림
4. 차트를 컴포넌트화
5. 컴포넌트의 플러그인화
6. 컴포넌트 통신을 이용한 차트 컴포넌트 기능 결합
```
